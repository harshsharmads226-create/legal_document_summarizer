# Legal Decision Summarizer

A web-based application that summarizes lengthy legal judgments into concise and meaningful summaries using NLP techniques.

## 🚀 Features
- Upload legal judgment PDFs
- Extracts and preprocesses legal text
- Generates structured summaries focused on judgments and key decisions
- Simple and clean UI
- Built using Python and Flask

## 🛠 Tech Stack
- Python
- Flask
- NLP (Text Processing)
- HTML, CSS, JavaScript

## 📌 Use Case
Helps law students, researchers, and professionals quickly understand long legal decisions.

## ▶️ How to Run
1. Clone the repository  
2. Install dependencies  
   `pip install -r requirements.txt`
3. Run the app  
   `python app.py`
4. Open browser at `http://127.0.0.1:5000/`

## 📤 Output
Concise legal summary highlighting judgments and important points
